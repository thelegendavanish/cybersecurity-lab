# Interactive Cyber Security Labs

A comprehensive, interactive educational platform designed to simulate various cyber threats and teach prevention strategies.

## 🚀 Overview
This repository contains a series of interactive labs developed with a modern, retro-futuristic dark/neon aesthetic. Each lab provides a hands-on simulation of a specific cybercrime or security concept, followed by educational insights and forensic detection techniques.

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Design**: Retro-futuristic theme, glassmorphism, neon accents
- **Typography**: Outfit, Space Grotesk, Space Mono (Google Fonts)
- **Visuals**: CSS Animations, SVG Filters, Dynamic Glow Effects

## 🧪 Interactive Labs

### Technical Attacks
- **Cyber Terrorism**: Simulates infrastructure disruption and computer-based weapons.
- **Phishing**: Realistic simulation of email credential theft and defensive controls.
- **Denial of Service (DoS)**: Visualize server crashes and latency spikes under DDoS volume.
- **Cross Site Scripting (XSS)**: Hands-on lab for injecting and executing malicious scripts.
- **Logic Bombs**: Demonstrates triggering malicious functions under specific logic conditions.
- **Computer Hacking**: Exploring vulnerability exploitation in a simulated console.
- **Computer Vandalism**: Visualizing the modification and destruction of system data.

### Financial & Legal
- **Online Auction Fraud**: Simulation of deceptive tactics used in auction settings.
- **Cyber-squatting**: Domain name trafficking and trademark profit simulation.
- **Salami Attack**: Step-by-step accumulation of wealth via minor invisible transactions.
- **Data Diddling**: Unauthorized altering of data before or during system entry.
- **Email Spoofing**: Creating forgeries of sender addresses to mislead recipients.
- **Internet Time Theft**: Resource/bandwidth theft simulation and impact visualization.
- **Spamming**: Launching bulk unsolicited data campaigns to observe network congestion.
- **Forgery & Counterfeiting**: Digital ID replication using layered security and forensic scanning tools (UV/Magnifier).

## 🏁 Getting Started
1. Navigate to the `Labs/` directory.
2. Open `index.html` in any modern web browser (Chrome/Edge/Safari).
3. Use the interactive categories (Social, Technical, Financial) to launch each simulation.

## ✨ Features
- **Hands-on Simulations**: Interactive UI components that allow you to "perform" the concepts described.
- **Forensic Suite**: Specialized virtual tools (UV lamp, forensic magnifying lens) to identify fakes.
- **Educational Insights**: Deep dives into the history, legal consequences, and real-world impact of each crime.
- **Prevention Tips**: Practical advice on WPA3 encryption, MAC filtering, VPN usage, and more.
- **Consistency**: All labs share the "Digital Mint" / "Retro-Dark" aesthetic for a seamless experience.
