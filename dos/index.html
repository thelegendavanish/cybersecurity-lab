DoS vs DDoS: The Key Differences

💻

DoS (Denial of Service)

A single computer and internet connection are used to flood a targeted server with packets.

  • One Source
  • Easier to Block (IP Ban)
  • Lower Bandwidth
🕸️

DDoS (Distributed DoS)

Multiple computers (often a Botnet) are used to attack the target simultaneously from multiple locations.

  • Many Sources (Botnet)
  • Difficult to Distinguish
  • Massive Bandwidth


Defense & Mitigation

🛡️

WAF & Firewalls

Web Application Firewalls filter and monitor HTTP traffic to and from a web application.

☁️

CDN Scrubbing

Services like Cloudflare absorb massive traffic and "scrub" it to ensure only clean traffic reaches your server.

⚖️

Load Balancing

Distributing incoming network traffic across a group of backend servers to prevent any single server from becoming overwhelmed.
